HTML & CSS

Overview

HTML

HTML is the foundation behind all web pages. HTML is a simple language that consists of a series of tags. These tags are used to structure the contents of an HTML file. A typical website consists of many different HTML files.

CSS

CSS is the language used to style HTML content. CSS code is used to describe how page content should be presented in different cases. CSS code should be kept in separate files to split between page content and presentation.

Glossary

tag

All HTML files live within an over-arching html tag. This is the basic tag that defines an html document.

id

An HTML element can have an id attribute to identify it.

class

HTML elements can have one or more classes, separated by spaces. You can style elements by using . and class name

Useful code snippets

HTML | Simple document
<!DOCTYPE html>

<html lang="en">

<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <title>page title</title>

    <link rel="stylesheet" href="css/style.css">
</head>

<body>
    <div class="container">
        <h1>hello world!</h1>
    </div>
</body>

</html>
CSS | Set box-model to border-box
*,
*:before,
*:after {
  -webkit-box-sizing: border-box;
     -moz-box-sizing: border-box;
          box-sizing: border-box;
}
CSS | Media query
@media (max-width: 600px) {

    display: none;
  }
